Layering Contract

CANVAS

One canvas. Every asset is authored on the same 64×64 with the same margins — base, cloak, totem, crown, all of it.

ANCHOR

The anchor is law. Align every overlay's base to ⟨32, 58⟩. Never nudge it per-asset or the whole set drifts.

LIGHT

One light. Upper-left, single source, on every layer — otherwise gear reads as pasted on rather than worn.

SYMMETRY

Mind the seam. No true center pixel on an even canvas — make central features 2 px wide, or commit to a consistent 1 px lean.

GROWTH

Margins are reserved. The bare piece must not fill them — leave headroom & side-room for crowns, cloaks, and staves.

SILHOUETTE

Reads at 1×. Check the shape at actual size. If it turns to mud, simplify the form before adding any detail.

PALETTE

Locked ramp. Pull every pixel from the six values above — new gear introduces no new greys, only recolours later.

SHOULDER

Optional second ref. If cloaks need a hook, fix one shoulder line now and reuse it — but feet stay the master.
