2.5D tactical RPG conversion of the owner's live TTRPG campaign (reality-TV dungeon
crawler — abducted contestants fight on an alien broadcast). **Read this first, then

5. **`docs/rules-addendum.md`** — canonical DIGITAL rulings where the TTRPG book is silent

   superseded by DIRECTION.md's ladder).

## Hard rules (from the architecture doc + direction)

- `simulation/` is headless MODEL: `extends RefCounted`, no Godot node deps, fully
  `to_dict()`/`from_dict()` serializable, **command-stream only** — state must be a pure
  function of (seed, ordered command log); no wall-clock reads or unlogged randomness.
- Only `controller/dal.gd` touches SQLite; only `controller/save_manager.gd` touches save
  files; scenes are presentation only and talk through `GameController` signals.
- No to-hit rolls — requirements auto-succeed; Forced Action d6 handles the failure path.
- Bosses need discoverable win conditions, never raw damage races.
- Vocabulary: rulebook conditions (bleeding/crushed/suffocation/chilled/exhausted/
  infected/burn/poison/dissolution) + this repo's seed enums. Do NOT import the
  character-sheet app's DMG_TYPES/RACES lists (known drift — see review-1).

  never break it from here.

## Running tests

`bash scripts/run_sim_tests.sh` — runs the headless sim test suite; requires a Godot 4.5
binary (`godot` on PATH or `$GODOT_BIN`). The script prints an explicit SKIP (exit 3) when
no binary is available — a SKIP is not a pass; record it honestly.
